This is a simple C/C++ program whose purpose is to calculate the roots of any polynomial function provided by the user using Newton's method.

Este é um simples programa C/C++ cujo objetivo é calcular as raízes de qualquer função polinomial fornecida pelo usuário usando o método de Newton.


Attention, don't use spaces when typing the function. To perform an exponentiation, use the symbol '^'.

Atenção, não utilizar espaços ao digitar a função. Para realizar uma potenciação, utilize o símbolo '^'.
